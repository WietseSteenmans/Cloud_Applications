Getting Started!

Create Some Courses

So you are new here, aren't you? So let's get started if you go to the tab "My Lessons" you can see it is empty except for one tile. You can use this tile to add a new course, just press the "Add Course" then you will see a textbox appear, type in the name and hit that add button. On this page you can add and delete courses with all of their content. You just added a course but if you press it, nothing interesting. Let's change that go to the "New Questions" tab. Here you can add new questions to a course. A multiple choice template is standard selected. If you'd like to create a yes or no question hit that dropdown box on the top and select it. On to the coursename you can type in a new one or select the one you created before in the dropdown box next to it. Fill in the other text area's and save the question if you are done. Hit the "Another Question" button if you'd like to create another one. That were the first baby steps. Let's run!
